# ContactStoreTest

   

#### Installing ContactStoreTest

         
```
git https://github.com/appasaheb4/ContactStoreTest.git
cd ContactStoreTest
sudo yarn install
# for android
npm run android
# for ios
react-native ios
```   
    
  
Make sure you have a `.env.dev` similar to `.env.example` in your project's root directory before running ContactStoreTest.
